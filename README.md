<h1 align="center">URL Shortener</h1>

## About

URL Shortener is an application for short links.

This Project was developed for Bootcamp in Digital Innovation One.

## Technologies

- NodeJs
- MongoDB

## Run Project

Clone Project

```git
git clone https://github.com/thiagotrs/url-shortener.git
```

API

```ssh
npm install
npm start
```

URL
```
http://localhost:5000/
```
